HyperText Markup Language

Hvad er HTML

HTML er et opmærkningssprog, der primært har til formål at strukturere indholdet på en hjemmeside og få tekst og billeder til at virke som links til andre html-sider.

Markup betyder at opmærke. I hjemmesidesammenhæng vil det sige at html definerer indhold på websider, f.eks. kan en tekst markeres som overskrift eller brødtekst, arrangeres i en punktopstilling eller tabel og et hyperlink kan indsættes som klikbar tekst.

Dermed kan webbrowsere som f.eks. Firefox, MS Internet Explorer, Safari, Opera eller Google Chrome, skelne de forskellige teksttyper fra hinanden og vise indholdet formateret efter hvilken type det er markeret som. Typisk blå skrift.

Hvad med udseende?

HTML indeholder mulighed for med ”CSS” at bestemme mere detaljeret over udseendet af f.eks. overskrifter, tabeller, hyperlink og indlejring af billeder. Html-filer bruges på internettet, især på World Wide Web. Html filer har Extension .Html eller .Htm.

HTML består af Elementer
Elementer opmærkes med "HTML - Tags".

HTML består af Elementer og der er omkring 120 stk. pr. ultimo 2023. Et eksempel på et HTML Element kunne være Article, - Altså en blok af sammenhængende tekst, der giver mening og kan læses for sig selv. F.eks. <strong>Aab ryger ud af Superligaen</strong> :0(

Nogle Elementer er forbeholdt eller beregnet til at indeholde bestemt indhold. F.eks. En Navigation, Billede eller Grafik, En sektion eller artikel på siden eller f.eks. Video-Tagget der er beregnet til at indeholde videoer!

Styling af HTML Elementer

Browsere har en bestemt måde at vise elementer på, afhængigt af hvilken type element det er. Disse måder at blive vist på og placere sig på er overordnet anbefalet af W3.org (World Wide Web Consortium), og det er disse indstillinger vi ændrer via CSS.
For at ændre den måde elementer vises på i browseren anvender vi CSS, (Cascading Style Sheets). CSS ændrer altså på den måde et element vises eller opfører sig på i Browseren, eksempelvis farve, størrelse, placering (Display og positioning), og meget mere.

3 måder at style på

Du kan anvende traditionel styling på 3 forskellige måder! - Inline - Styling hvor styling bliver påført elementerne direkte, Inline Stylesheet En overordnet liste af styling i dokumentets Head - tag og Ekstern Stylesheets (Eksterne Fælles styling).

Search engine optimization (SEO) is the process of growing a website’s organic search traffic. It’s where you do things that help you show up and rank higher in a search engine’s organic results.

Meta - Tags?
Metadata is "data (information) about data".
The "meta-tag" defines metadata about an HTML document. Metadata is data (information) about data. Meta-tags always go inside the "head - Element", and are typically used to specify character set, page description, keywords, viewport settings and more.
Metadata will not be displayed on the page, but is machine parsable. Metadata is used by browsers (how to display content or reload page), search engines (keywords), and other web services. There is a method to let web designers take control over the viewport (the user's visible area of a web page), through the "meta-tag".
The presence of Artificial Intelligence and "Machinelearning" in browsers like Google and Bing leaves some of these "SEO - Meta-tags" somewhat unnecessary, or at least "Less important" these days? - Example: "Content", "Description" and "Language".
